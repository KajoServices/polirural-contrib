# PoliRural Project: contribs

http://polirural.eu/

## Requirements
* Python 3.5+

## Setup environment

    $ python3 -m venv pr_contrib
    $ source pr_contrib/bin/activate
    $ pip install -r requirements.txt

## Usage

### Parsers

Run for details:

    $ python parsers/thesaurus.py --help
    $ python parsers/keywords.py --help
